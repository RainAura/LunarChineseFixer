# 更新记录

## 1.0.7 - 2026-07-13

- 移除 GitHub 按钮外框；
- 将 GitHub 图标和文字作为整体水平、垂直居中。

## 1.0.6 - 2026-07-13

- 将目录提示统一改为“请选择 Lunar 客户端的我的世界路径”；
- 同步优化选择按钮、目录选择窗口和未检测状态的文字表达。
- 修复高 DPI 缩放时路径选择按钮和 GitHub 按钮移出可视区域；
- 标题栏移除作者名，并在 GitHub 按钮旁显示“作者 RainAura & Codex”。

## 1.0.5 - 2026-07-13

- 完全重做窗口排版，改为状态、配置、操作、日志四段纵向结构；
- 移除原有深蓝配色、圆角卡片和渐变按钮，采用简洁的黑白灰浅色界面；
- 重新分配各区域高度和间距，解决文字与边框拥挤问题；
- 保留固定窗口尺寸及右上角 GitHub 图标入口。
- 增加 .NET Framework 4.8 轻量构建，显著缩小 EXE 体积。

## 1.0.4 - 2026-07-13

- 重做为更小、更轻量的固定尺寸单页界面；
- 移除占空间的统计卡片，大幅缩短配置路径区域；
- 在标题区域加入始终可见的 GitHub 图标按钮；
- 修复单文件发布时 GitHub 图标资源丢失。

## 1.0.3 - 2026-07-13

- 窗口缩小为 920×650，并禁止拖动改变尺寸；
- 移除顶部渐变装饰，整体视觉更加简洁；
- GitHub 文字按钮替换为官方图标，点击图标打开项目仓库；
- 调整状态区、统计卡片和内容区间距；
- 移除组合窗口样式，配合固定尺寸减少重绘残影。

## 1.0.2 - 2026-07-13

- 配置列表不再显示用户本地路径；
- 增加始终可见的“选择 .minecraft 文件夹”按钮和明确的目录说明；
- 增加 GitHub 项目主页按钮；
- 放宽窗口、卡片、列表与操作区间距；
- 增加组合重绘缓冲，减少窗口拖动和缩放残影；
- 标题栏改为显示应用内嵌 Lunar 图标；
- 输出文件名改为 `LunarChineseFixer.exe`，避免旧文件图标缓存。

## 1.0.1 - 2026-07-13

- 修复程序启动时 SplitContainer 尺寸尚未初始化导致的崩溃；

- 应用图标替换为 Lunar Client 图标。

## 1.0.0 - 2026-07-13

- 自动识别 Lunar Client 1.8 隔离配置；
- 自动识别默认 `.minecraft` 与 Lunar 日志中的自定义 `gameDir`；
- 增加手动选择游戏目录功能；
- 增加 Lunar 游戏运行状态保护；
- 增加一键修复与恢复功能；
- 增加扫描结果、诊断日志和 UTF-8 报告导出；
- 提供 Windows x64 自包含单文件 EXE；
- 增加 GitHub Actions 自动测试与构建。
